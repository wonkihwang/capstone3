# 📊 Skill Gap Analyzer + ML Role Predictor

This project is an interactive web application that helps users evaluate their current skills and identify gaps based on real data science job market trends.

🔗 Live App: https://skgap-analyzer.streamlit.app/

---

## 🚀 Features

- Select your current skills using an intuitive UI  
- Get a match score for your target role  
- Receive recommended skills to improve  
- Machine Learning model predicts the most suitable role  
- Displays confidence levels for predictions  

---

## 🧠 Technologies Used

- Python  
- Pandas  
- Scikit-learn (Random Forest)  
- Streamlit  

---

## 📊 How It Works

1. User selects their skills  
2. System compares with market-required skills  
3. Machine learning model predicts role suitability  
4. Outputs:
   - Match score  
   - Recommended skills  
   - Predicted role with confidence  

---

## 🎯 Purpose

This project demonstrates how data analysis and machine learning can be applied to a real-world problem like career guidance.
